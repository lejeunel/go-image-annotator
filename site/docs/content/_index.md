# Introduction

This application aims to organize and annotate image data.

# Content

- [Development]( {{% relref "dev" %}} ) explains how to contribute to this project.
- [Architecture]( {{% relref "architecture" %}} ) describe the domain-objects
    that we leverage, and show how they are inter-connected.
- [Tutorials]( {{% relref "tutorials/00-onboarding" %}} ) provide a 
    walkthrough to help you get up to speed with coding new features.
    We also give a short description of the Model-View-Controller architecture,
    which we use in this app.
- [Software Development Kit (SDK)]( {{% relref "sdk" %}} ) introduces a package
    to help manage resources remotely and integrate with python pipelines.
- [Workflows]( {{% relref "workflows" %}} ) show how to solve
    common use-cases through the API, such as annotating images and ingesting
    new data.
